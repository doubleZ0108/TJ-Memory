第三部分是我这个项目的有一个新颖的地方，全景的世界

1. 我们之前用文字的博客记录生活，现在慢慢我们用朋友圈，图片博客的形式甚至是视频播客vlog记录生活，这里我大胆的提出了一个全景博客的概念
2. 我可以拖动鼠标进行视角的变换，这里用了图形学和计算机视觉的相关算法进行建模

6. 我们再看几张图，还有一些高清的图片
